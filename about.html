About

This application was created with the intention to help people remember to drink water and keep a record of their progress. Moreover, things in this world are more difficult, if done alone. Thus, we decided to create a community in which people can help each other reach their desireable goal of drinking the right amount of water. Help us grow this community.

Even though this is an application, this is only a project where I can show what I am learning in my C S 260 Web Programming class. This is only for non-profit educational purposes. No part of this code or program should be used outside of that definition.
